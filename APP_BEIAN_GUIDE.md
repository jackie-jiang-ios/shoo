# 🛡️ 防兽神器（Shoo）- APP备案申请指南（阿里云）

## 一、APP备案前置条件

在开始之前，请确保你已准备好以下材料：

| # | 材料 | 说明 |
|---|------|------|
| 1 | ✅ 阿里云账号 | 已注册并完成实名认证（个人或企业） |
| 2 | ✅ 阿里云服务器 | 至少一台中国内地节点的 ECS 实例（用于获取备案服务码） |
| 3 | ✅ 域名 | 已注册并在阿里云做过实名认证的域名 |
| 4 | ✅ APP图标 | 1024x1024 的应用图标 |
| 5 | ✅ 身份证 | 主体负责人身份证正反面照片 |
| 6 | ✅ 手机号 | 备案负责人手机号（需保持畅通） |

> 💡 如果是个人备案，主体负责人就是你自己；如果是企业备案，需要营业执照。

---

## 二、阿里云APP备案完整步骤

### 步骤1️⃣：登录阿里云备案系统

1. 打开浏览器，访问阿里云备案中心：
   👉 **https://beian.aliyun.com**
2. 使用你的阿里云账号登录
3. 进入「我的备案」页面

---

### 步骤2️⃣：获取备案服务码

> 📌 备案服务码用于关联你的阿里云服务器，每个APP需要一个服务码。

1. 在备案页面，找到「备案服务码管理」
2. 点击「申请服务码」
3. 选择你的阿里云 ECS 实例
4. 系统自动生成备案服务码

> 💡 一台 ECS 实例可申请 5 个备案服务码，每增加一台 ECS 可多 5 个。

---

### 步骤3️⃣：新增APP备案

1. 在「我的备案」页面，点击 **「新增APP」** 按钮
2. 弹出输入框，输入APP名称：**防兽神器**
3. 点击「确认」

---

### 步骤4️⃣：填写APP基本信息

根据页面提示填写以下信息：

| 字段 | 填写内容 | 是否必填 |
|------|---------|---------|
| **APP图标** | 上传 1024x1024 图标 | ✅ 必填 |
| **APP名称** | 防兽神器 | ✅ 必填 |
| **云服务IP地址** | 你的阿里云ECS公网IP | ✅ 必填 |
| **内容分类** | 工具 / 实用工具 | ✅ 必填 |
| **前置审批** | 无（不涉及新闻/出版/教育等） | 选填 |
| **APP语言** | 中文 | ✅ 必填 |
| **备注** | 一款通过声音驱赶野兽的多平台实用工具应用 | 选填 |
| **是否对外提供SDK** | 否 | ✅ 必填 |
| **使用的三方SDK** | 可不填 | 选填 |
| **APP后台服务域名** | 你的服务器域名，如 api.shoo-app.com | ✅ 必填 |
| **使用阿里云服务器的域名** | 同上 | ✅ 必填 |

---

### 步骤5️⃣：填写平台特征信息

#### 🤖 Android 平台（如需备案安卓端）

| 字段 | 说明 | 如何获取 |
|------|------|---------|
| **软件包名称** | 如 `com.shoo.app` | 自定义，与项目包名一致 |
| **公钥** | 签名证书的公钥 | 见下方获取方法 |
| **证书MD5指纹** | 签名证书的MD5 | 见下方获取方法 |
| **服务器域名** | APP通信的服务器域名 | 你的API域名 |

**获取安卓公钥和MD5指纹：**

```bash
# 方法一：使用 keytool 命令（已有 keystore 证书）
keytool -list -v -keystore your_keystore.keystore

# 方法二：使用 keytool 导出证书后查看
keytool -export -alias your_alias -file certificate.cer -keystore your_keystore.keystore
# 然后双击 certificate.cer 文件查看公钥

# 方法三：阿里云自带检测工具
# 在备案页面点击「上传检测」→ 上传APK → 自动填充
```

#### 🍎 iOS 平台（如需备案苹果端）

| 字段 | 说明 | 如何获取 |
|------|------|---------|
| **Bundle ID** | 如 `com.shoo.app` | Xcode 中查看 / Apple Developer 后台 |
| **公钥** | iOS 分发证书的公钥 | 见下方获取方法 |
| **证书SHA-1指纹** | iOS 分发证书的SHA-1 | 见下方获取方法 |
| **服务器域名** | APP通信的服务器域名 | 你的API域名 |

**获取iOS公钥和SHA-1指纹：**

```bash
# 方法一：从 Apple Developer 后台下载证书后
# 将 .cer 证书导入钥匙串，然后导出 .p12 文件
# 使用 openssl 查看
openssl pkcs12 -in your_cert.p12 -nocerts -out key.pem
openssl rsa -in key.pem -pubout -out public_key.pem
openssl x509 -in your_cert.cer -inform DER -fingerprint -sha1

# 方法二：使用 AppUploader 工具
# 下载地址：https://applicationloader.net
# 登录后 → 文件查看页面 → 可直接复制公钥和SHA1信息
```

---

### 步骤6️⃣：上传主体资料

根据提示上传以下材料：

| 材料 | 要求 |
|------|------|
| 主体负责人身份证正面 | 清晰彩色照片/扫描件 |
| 主体负责人身份证反面 | 清晰彩色照片/扫描件 |
| 手持身份证照片 | 部分省份要求（按提示） |
| 域名证书 | 阿里云域名管理中下载 |
| 企业营业执照 | 仅企业备案需要 |

> 📱 上传资料需要使用 **阿里云APP** 扫码在手机端操作，不支持PC端直接上传。

---

### 步骤7️⃣：阿里云初审

1. 提交备案后，阿里云通常在 **1个工作日** 内完成初审
2. 初审期间阿里云客服可能会打电话核实信息，**请保持手机畅通**
3. 如有问题，按要求修改后重新提交

---

### 步骤8️⃣：管局审核

1. 阿里云初审通过后，资料提交至 **工信部管局** 审核
2. 管局审核期间会发送 **短信核验**，收到短信后需在 24 小时内完成验证
   - 访问：https://beian.miit.gov.cn
   - 输入短信验证码 + 手机号 + 身份证后6位
3. 管局审核周期通常 **5-20个工作日**（各省份不同）

---

### 步骤9️⃣：备案成功

1. 审核通过后，会收到短信和邮件通知
2. 在阿里云备案页面查看备案号
3. 备案号格式示例：**京ICP备2024XXXXXX号-X**

---

## 三、备案成功后还需要做的事

### 3.1 公安联网备案（⚠️ 必须做）

APP开通之日起 **30日内**，需完成公安联网备案：

1. 访问 **全国互联网安全管理服务平台**
   👉 https://www.beian.gov.cn
2. 注册个人账号
3. 选择「APP备案」→ 填写APP信息
4. 提交审核（通常 3-5 个工作日）

### 3.2 备案号展示

APP需在以下位置展示备案号：
- APP的「关于」页面
- APP官网页面底部

### 3.3 后续变更

如以下信息变更，需在 **10个工作日** 内更新备案：
- APP名称变更
- 主体信息变更
- 服务器IP/域名变更

## 四、常见问题

| 问题 | 答案 |
|------|------|
| APP还没开发完可以备案吗？ | ✅ 可以，备案不要求APP已上架 |
| 一个备案主体可以备多个APP吗？ | ✅ 可以，没有数量限制 |
| 个人可以备案APP吗？ | ✅ 可以，个人和企业都能备案 |
| 备案需要多长时间？ | 阿里云初审1天 + 管局5-20天 ≈ **6-21天** |
| 备案收费吗？ | ❌ 阿里云备案免费，但需要有阿里云服务器 |
| 没有域名可以备案吗？ | ❌ 必须有已实名的域名 |
| 安卓和iOS要分别备案吗？ | 不需要，一个APP备案可同时覆盖多平台，填写时勾选即可 |
| 手表端APP需要单独备案吗？ | 目前手表端APP如果与手机端是同一应用，不需要单独备案 |
